# travel-server
여행 일정 서비스 서버. (가칭) 

----------------------------------

### 사용 기술
- JDK 17
- spring boot 3.1.1
- lombok 1.18.30
- swagger v3
- MySQL 8.0
- gradle
- Docker

### 참고 링크

- [이슈](https://github.com/DENEB1593/travel-server/issues)

### 실행
    ./gradlew bootRun 
    ※ 사전에 Docker 설치 필수

### API 확인
    localhost:8080/swagger-ui/index.html
